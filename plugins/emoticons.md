// adds toolbar control

The Emoticons plugin adds a toolbar control that let's users insert smiley images into TinyMCE's editable area.

**Type:** `String`

##### Example

```js
tinymce.init({
  selector: "textarea",  // change this value according to your html
  plugins: "emoticons",
  toolbar: "emoticons"
});
```
