Root level namespace this contains classes directly releated to the TinyMCE editor.

Namespaces

Namespace Summary
tinymce.dom Contains classes for handling the browsers DOM.
tinymce.html Contains html parser and serializer logic.
tinymce.ui Contains the different UI types such as buttons, listboxes etc.
tinymce.util Contains various utility classes such as json parser, cookies etc.

Classes

Class Summary
tinymce.AddOnManager This class handles the loading of themes/plugins or other add-ons and their language packs
tinymce.Theme TinyMCE theme class
tinymce.Plugin Plugin base class, this is a pseudo class that describes how a plugin is to be created for TinyMCE
tinymce.Event This is the base class for all TinyMCE events
tinymce.ContentEvent This class is the event object send when the content events occurs such as GetContent/SetContent
tinymce.CommandEvent This class is the event object send when the BeforeExecCommand and ExecCommand event occurs
tinymce.ProgressStateEvent This class is the event object send when the ProgressState event occurs
tinymce.FocusEvent This class is the event object sent when editors are focused/blurred
tinymce.ResizeEvent This class is the event object sent when objects gets resized within the editor
tinymce.Editor This class contains the core logic for a TinyMCE editor
tinymce.EditorCommands This class enables you to add custom editor commands and it contains overrides for native browser co
tinymce.EditorManager This class used as a factory for manager for tinymce
tinymce.Env This class contains various environment constants like browser versions etc
tinymce.FocusManager This class manages the focus/blur state of the editor
tinymce.Formatter Text formatter engine class
tinymce.Shortcuts Contains all logic for handling of keyboard shortcuts
tinymce.UndoManager This class handles the undo/redo history levels for the editor
tinymce.WindowManager This class handles the creation of native windows and dialogs
