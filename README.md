## TinyMCE documentation

### First time setup

    $ git clone git@github.com:tinymce/tinymce-docs.git
    $ cd tinymce-docs
    $ sudo gem install bundle
    $ ./init.sh

### Run development version

    $ ./serve.sh

Navigate to http://127.0.0.1:4000/
