--- layout: default title: tinymce.UndoManager.transact ---
public function transact(callback:function):void
Executes the specified function in an undo transation. The selection before the modification will be stored to the undo stack and if the DOM changes it will add a new undo level. Any methods within the transation that adds undo levels will be ignored. So a transation can include calls to execCommand or editor.insertContent.

Params

Name Type Description
callback function Function to execute dom manipulation logic in.
