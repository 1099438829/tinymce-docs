public function prop(name:String, value:Object):tinymce.ui.Collection
Sets/gets the specific property on the items in the collection. The same as executing control.<property>(<value>);

Params

Name Type Description
name String Property name to get/set.
value Object Optional object value to set.

Returns

tinymce.ui.Collection Current collection instance or value of the first item on a get operation.
