public function create(settings:Object/String):tinymce.ui.Control
Creates a new control instance based on the settings provided. The instance created will be based on the specified type property it can also create whole structures of components out of the specified JSON object.

Examples

tinymce.ui.Factory.create({
    type: 'button',
    text: 'Hello world!'
});

Params

Name Type Description
settings Object String Name/Value object with items used to create the type.

Returns

tinymce.ui.Control Control instance based on the specified type.
