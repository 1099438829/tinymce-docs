--- layout: default title: tinymce.Formatter.getCssText ---
public function getCssText(format:String/Object):String
Returns a preview css text for the specified format.

Examples

var cssText1 = editor.formatter.getCssText('bold');
var cssText2 = editor.formatter.getCssText({inline: 'b'});

Params

Name Type Description
format String Object Format to generate preview css text for.

Returns

String Css text for the specified format.
