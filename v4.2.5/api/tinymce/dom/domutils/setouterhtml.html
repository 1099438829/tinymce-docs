public function setOuterHTML(elm:Element/String/Array, html:Object, doc:Document):void
Sets the specified outer HTML on an element or elements.

Examples

// Sets the outer HTML of all paragraphs in the active editor
tinymce.activeEditor.dom.setOuterHTML(tinymce.activeEditor.dom.select('p'), '<div>some html</div>');

// Sets the outer HTML of an element by id in the document
tinymce.DOM.setOuterHTML('mydiv', '<div>some html</div>');

Params

Name Type Description
elm Element String Array DOM element, element id string or array of elements/ids to set outer HTML on.
html Object HTML code to set as outer value for the element.
doc Document Optional document scope to use in this process - defaults to the document of the DOM class.
