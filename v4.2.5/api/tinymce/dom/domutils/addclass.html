public function addClass(elm:String/Element/Array, cls:String):String/Array
Adds a class to the specified element or elements.

Examples

// Adds a class to all paragraphs in the active editor
tinymce.activeEditor.dom.addClass(tinymce.activeEditor.dom.select('p'), 'myclass');

// Adds a class to a specific element in the current page
tinymce.DOM.addClass('mydiv', 'myclass');

Params

Name Type Description
elm String Element Array Element ID string or DOM element or array with elements or IDs.
cls String Class name to add to each element.

Returns

String Array String with new class value or array with new class values for all elements.
