public function css(name:String/Object, value:String):tinymce.dom.DomQuery/String
Sets/gets styles on the elements in the current set.

Params

Name Type Description
name String Object Name of style to get or an object with styles to set.
value String Optional value to set.

Returns

tinymce.dom.DomQuery String Current set or the specified style when only the name is specified.
