public function offset(offset:Object):Object/tinymce.dom.DomQuery
Returns the offset of the first element in set or sets the top/left css properties of all elements in set.

Params

Name Type Description
offset Object Optional offset object to set on each item.

Returns

Object tinymce.dom.DomQuery Returns the first element offset or the current set if you specified an offset.
