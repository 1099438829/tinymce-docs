public function add(id:String, addOn:tinymce.Theme/tinymce.Plugin):tinymce.Theme/tinymce.Plugin
Adds a instance of the add-on by it's short name.

Examples

// Create a simple plugin
tinymce.create('tinymce.plugins.TestPlugin', {
  TestPlugin: function(ed, url) {
  ed.on('click', function(e) {
     ed.windowManager.alert('Hello World!');
  });
  }
});

// Register plugin using the add method
tinymce.PluginManager.add('test', tinymce.plugins.TestPlugin);

// Initialize TinyMCE
tinymce.init({
 ...
 plugins: '-test' // Init the plugin but don't try to load it
});

Params

Name Type Description
id String Short name/id for the add-on.
addOn tinymce.Theme tinymce.Plugin Theme or plugin to add.

Returns

tinymce.Theme tinymce.Plugin The same theme or plugin instance that got passed in.
