public function load(name:String, addOnUrl:String, callback:function, scope:Object):void
Loads an add-on from a specific url.

Examples

// Loads a plugin from an external URL
tinymce.PluginManager.load('myplugin', '/some/dir/someplugin/plugin.js');

// Initialize TinyMCE
tinymce.init({
 ...
 plugins: '-myplugin' // Don't try to load it again
});

Params

Name Type Description
name String Short name of the add-on that gets loaded.
addOnUrl String URL to the add-on that will get loaded.
callback function Optional callback to execute ones the add-on is loaded.
scope Object Optional scope to execute the callback in.
