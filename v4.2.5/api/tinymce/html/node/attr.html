public function attr(name:String, value:String):String/tinymce.html.Node
Gets/sets or removes an attribute by name.

Examples

someNode.attr("name", "value"); // Sets an attribute
console.log(someNode.attr("name")); // Gets an attribute
someNode.attr("name", null); // Removes an attribute

Params

Name Type Description
name String Attribute name to set or get.
value String Optional value to set.

Returns

String tinymce.html.Node String or undefined on a get operation or the current node on a set operation.
