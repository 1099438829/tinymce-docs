--- layout: default title: tinymce.Editor.BeforeSetContent ---
public event BeforeSetContent(e:tinymce.ContentEvent)
Fires before contents is inserted into the editor.

Examples

tinymce.init({
    ...
    setup: function(editor) {
        editor.on('BeforeSetContent', function(e) {
            console.log('BeforeSetContent event', e);
        });
    }
});

Params

Name Type Description
e tinymce.ContentEvent Event arguments.
