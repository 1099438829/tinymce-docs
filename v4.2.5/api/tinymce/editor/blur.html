--- layout: default title: tinymce.Editor.blur ---
public event blur(e:tinymce.FocusEvent)
Fires when the editor is blurred.

Examples

tinymce.init({
    ...
    setup: function(editor) {
        editor.on('blur', function(e) {
            console.log('blur event', e);
        });
    }
});

Params

Name Type Description
e tinymce.FocusEvent Event arguments.
