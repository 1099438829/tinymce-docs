public function addContextToolbar(predicate:function/string, items:String/Array):void
Adds a contextual toolbar to be rendered when the selector matches.

Params

Name Type Description
predicate function string Predicate that needs to return true if provided strings get converted into CSS predicates.
items String Array String or array with items to add to the context toolbar.
