--- layout: default title: tinymce.Editor.LoadContent ---
public event LoadContent(e:tinymce.ContentEvent)
Fires after contents has been loaded into the editor.

Examples

tinymce.init({
    ...
    setup: function(editor) {
        editor.on('LoadContent', function(e) {
            console.log('LoadContent event', e);
        });
    }
});

Params

Name Type Description
e tinymce.ContentEvent Event arguments.
