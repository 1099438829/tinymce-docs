--- layout: default title: tinymce.Editor.ProgressState ---
public event ProgressState(e:tinymce.ProgressStateEvent)
Fires when a progress event is made. To display a throbber/loader.

Examples

tinymce.init({
    ...
    setup: function(editor) {
        editor.on('ProgressState', function(e) {
            console.log('ProgressState event', e);
        });
    }
});

Params

Name Type Description
e tinymce.ProgressStateEvent Event arguments.
