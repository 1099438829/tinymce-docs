--- layout: default title: tinymce.ui.ReflowQueue ---
This class will automatically reflow controls on the next animation frame within a few milliseconds on older browsers. If the user manually reflows then the automatic reflow will be cancelled. This class is unsed internally when various control states changes that triggers a reflow.
