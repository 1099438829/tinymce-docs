--- layout: default title: tinymce.dom.TreeWalker ---
TreeWalker class enables you to walk the DOM in a linear manner.

Examples

var walker = new tinymce.dom.TreeWalker(startNode);

do {
    console.log(walker.current());
} while (walker.next());
